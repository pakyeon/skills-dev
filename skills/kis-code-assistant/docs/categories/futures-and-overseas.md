# Futures And Overseas Guide

Load this guide only after search results land in:

- `domestic_futureoption`
- `overseas_stock`
- `overseas_futureoption`

## `domestic_futureoption`

Typical intents:

- domestic futures and options pricing
- asking price boards
- derivative balance and orders
- night-session or real-time derivatives data

## `overseas_stock`

Typical intents:

- US or Asia stock prices
- overseas balance and order history
- rights, news, rankings, and market analytics
- delayed or real-time quote flows

## `overseas_futureoption`

Typical intents:

- overseas futures pricing
- options asking price
- order and deposit queries
- live futures quote subscriptions

## Rule Of Thumb

If the user is asking for market execution or live environment access, stay in discovery/code-generation mode unless Trading MCP is available.
